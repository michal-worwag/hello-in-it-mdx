# What are pseudo-classes in CSS?

## Answer

Pseudo-classes define the special state of an element, like `:hover`, `:focus`, or `:nth-child`.

```css
a:hover {
  color: red;
}
```

[Learn more](https://developer.mozilla.org/en-US/docs/Web/CSS/Pseudo-classes)
