# How do you create a CSS Grid layout?

## Answer

CSS Grid allows for 2D layout control (rows and columns). Use `display: grid;`.

```css
.container {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 10px;
}
```
